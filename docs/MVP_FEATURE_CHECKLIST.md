# MVP 핵심 기능 검증 체크리스트

이 문서는 Pitch Camping 서비스의 MVP 핵심 기능 4개에 대한 검증 체크리스트입니다.

## MVP 핵심 기능 목록

1. 캠핑장 목록 + 지역/타입 필터
2. 네이버 지도 연동
3. 키워드 검색
4. 상세페이지

---

## 1. 캠핑장 목록 + 지역/타입 필터

### 1.1 기본 목록 표시

- [ ] 홈페이지 접속 시 캠핑장 목록이 표시됨
- [ ] 목록이 그리드 레이아웃으로 표시됨 (데스크톱)
- [ ] 목록이 세로 레이아웃으로 표시됨 (모바일)
- [ ] 각 카드에 다음 정보가 표시됨:
  - [ ] 썸네일 이미지
  - [ ] 캠핑장명
  - [ ] 주소
  - [ ] 캠핑 타입 뱃지
  - [ ] 시설 아이콘
- [ ] 이미지가 없는 경우 기본 이미지가 표시됨

### 1.2 지역 필터

- [ ] 지역 드롭다운이 표시됨
- [ ] "전체" 옵션 선택 시 모든 지역 표시
- [ ] 특정 지역 선택 시 해당 지역 캠핑장만 표시
- [ ] 필터 변경 시 URL 쿼리 파라미터 업데이트
- [ ] 페이지 새로고침 시 필터 상태 유지

### 1.3 캠핑 타입 필터

- [ ] 캠핑 타입 드롭다운이 표시됨
- [ ] "전체" 옵션 선택 시 모든 타입 표시
- [ ] 특정 타입 선택 시 해당 타입만 표시 (일반야영장, 자동차야영장, 글램핑, 카라반)
- [ ] 필터 변경 시 즉시 목록 업데이트

### 1.4 시설 필터

- [ ] 시설 체크박스가 표시됨
- [ ] 여러 시설을 동시에 선택 가능
- [ ] 선택된 시설이 있는 캠핑장만 표시
- [ ] 필터 초기화 버튼 동작 확인

### 1.5 정렬 옵션

- [ ] 정렬 드롭다운이 표시됨
- [ ] 이름순 정렬 동작 확인
- [ ] 지역순 정렬 동작 확인
- [ ] 정렬 변경 시 목록 업데이트

### 1.6 페이지네이션

- [ ] 페이지당 20개 항목 표시
- [ ] 이전/다음 버튼 동작 확인
- [ ] 페이지 번호 버튼 동작 확인
- [ ] 현재 페이지 하이라이트 표시
- [ ] 첫 페이지에서 이전 버튼 비활성화
- [ ] 마지막 페이지에서 다음 버튼 비활성화

### 1.7 로딩 및 에러 상태

- [ ] 데이터 로딩 중 스켈레톤 UI 표시
- [ ] API 에러 시 에러 메시지 표시
- [ ] 빈 결과 시 안내 메시지 표시
- [ ] 재시도 버튼 동작 확인

## 2. 네이버 지도 연동

### 2.1 지도 로드

- [ ] 지도가 정상적으로 로드됨
- [ ] 로딩 중 스켈레톤 또는 스피너 표시
- [ ] 지도 로드 실패 시 에러 메시지 표시
- [ ] 초기 중심 좌표가 한국 중심으로 설정됨

### 2.2 마커 표시

- [ ] 목록에 표시된 캠핑장이 지도에 마커로 표시됨
- [ ] 마커 클릭 시 인포윈도우 표시
- [ ] 인포윈도우에 캠핑장명, 주소, 상세보기 링크 표시
- [ ] 여러 마커가 지도에 정확히 표시됨

### 2.3 지도 이동 및 줌

- [ ] 지도 드래그로 이동 가능
- [ ] 줌 인/아웃 동작 확인
- [ ] 모든 마커가 보이도록 자동 범위 조정

### 2.4 리스트-지도 상호연동

- [ ] 리스트 카드 클릭 시 해당 마커로 지도 이동
- [ ] 지도 범위 자동 조정 (zoom level 15)
- [ ] 선택된 캠핑장의 인포윈도우 자동 열림
- [ ] 마커 클릭 시 해당 카드 하이라이트 (데스크톱)

### 2.5 반응형 레이아웃

#### 데스크톱 (> 1024px)
- [ ] 리스트와 지도가 나란히 표시됨 (2단 레이아웃)
- [ ] 지도가 sticky로 고정됨

#### 모바일 (< 1024px)
- [ ] 목록/지도 탭 전환 UI 표시
- [ ] 탭 전환 시 해당 화면만 표시됨
- [ ] 지도 높이가 적절함 (600px)

## 3. 키워드 검색

### 3.1 검색 입력

- [ ] 검색창이 표시됨
- [ ] 검색 아이콘 표시 확인
- [ ] 플레이스홀더 텍스트 표시 확인
- [ ] 입력 가능 확인

### 3.2 검색 실행

- [ ] Enter 키 입력 시 검색 실행
- [ ] 검색 버튼 클릭 시 검색 실행 (있는 경우)
- [ ] 검색어가 URL 쿼리 파라미터에 반영됨
- [ ] 검색 실행 시 로딩 상태 표시

### 3.3 검색 결과

- [ ] 검색 결과가 목록에 표시됨
- [ ] 검색 결과가 지도에 마커로 표시됨
- [ ] 검색 결과 개수 표시
- [ ] 검색어가 결과에 하이라이트됨 (선택 사항)

### 3.4 검색어 초기화

- [ ] X 버튼 클릭 시 검색어 초기화
- [ ] 검색어 초기화 시 전체 목록 표시
- [ ] URL 쿼리 파라미터에서 검색어 제거

### 3.5 검색 + 필터 조합

- [ ] 검색과 필터를 동시에 사용 가능
- [ ] 검색 결과에 필터 적용 가능
- [ ] 필터 변경 시 검색어 유지
- [ ] 검색 실행 시 필터 유지

### 3.6 빈 결과 처리

- [ ] 검색 결과가 없을 때 안내 메시지 표시
- [ ] 검색 조건 변경 안내
- [ ] 검색어 초기화 제안

## 4. 상세페이지

### 4.1 기본 정보 표시

- [ ] 캠핑장명 표시

- [ ] 전화번호 표시 (있는 경우)

- [ ] 운영 정보 표시
- [ ] 캠핑 타입 표시
- [ ] 시설 정보 표시

### 4.2 이미지 갤러리

- [ ] 대표 이미지 표시

- [ ] 이전/다음 버튼 동작 확인

### 4.3 공유 기능

- [ ] 공유 버튼 표시

- [ ] 복사 완료 토스트 메시지 표시

- [ ] 북마크 상태가 정확히 표시됨
- [ ] 페이지 새로고침 시 북마크 상태 유지

#### 비인증 사용자
- [ ] 북마크 버튼 표시
- [ ] 북마크 추가 시 localStorage에 저장
- [ ] 북마크 상태가 정확히 표시됨
- [ ] 로그인 후 북마크 동기화 (선택 사항)

### 4.5 리뷰 섹션

- [ ] 리뷰 섹션 표시
- [ ] 평균 평점 표시
- [ ] 리뷰 목록 표시
- [ ] 리뷰 작성 폼 표시 (인증된 사용자)
- [ ] 리뷰 작성 가능 (1-5점 평점)
- [ ] 리뷰 수정/삭제 가능 (작성자만)
- [ ] 리뷰 도움됨 기능 동작

- [ ] `camping_stats` 테이블에 반영
- [ ] 중복 조회 방지 (선택 사항)

### 4.7 네비게이션

- [ ] 뒤로가기 버튼 동작 확인
- [ ] 브레드크럼 네비게이션 (선택 사항)
